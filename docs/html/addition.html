addition module¶

addition.get_batch()¶

生成adding problem 数据集,每个批次大小为BATCH_SIZE,序列长度为TIME_STEPS,返回data

addition.indrnn_model(first_input_init, inputs_ph)¶

indrnn模型:搭建两层indrnn模型,每层神经元的数量为NUM_UNITS，两层总参数为：TIME_STEPS*NUM_UNITS+NUM_UNITS*NUM_UNITS+2*NUM_UNITS

addition.lstm_model(first_input_init, inputs_ph)¶

lstmrnn模型:搭建两层lstmrnn模型,每层神经元的数量为NUM_UNITS, 两层总参数为：TIME_STEPS*NUM_UNITS+3*NUM_UNITS

addition.main()¶

adding problem实验主程序,使用３种模型对数据集进行训练和测试,即indrnn,Irnn,lstm网络.

addition.rnn_model(first_input_init, inputs_ph)¶

lstmrnn模型:搭建两层lstmrnn模型,每层神经元的数量为NUM_UNITS, 两层总参数为：TIME_STEPS*NUM_UNITS+3*NUM_UNITS,激活函数用relu代替sigmod函数和tanh函数
